LineLearner tutorial

To add lines, you can:
  Use the textarea to add a single line.
  Use the textbox to add multiple lines, separated by newlines.
      
Once lines are added, you can manage them:
  Click Show New Line to display the lines.
  Click Subtract Line to hide the last displayed line.
  Click Subtract All Lines to hide all displayed lines.
      
When a line is shown:
  Click Remove Line to delete it permanently.
  Click Edit Line to modify it.
